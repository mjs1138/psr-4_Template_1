The key to Composer-Autoload is the Key "autoload" and its associated Value in the projects composer.json file. Example:


  "autoload": {
    "psr-4": {
      "ClassSrcFolder_TopLevelNamespace\\": "app/ClassSrcFolder/"
    }
  }

Composer uses this value to setup/control the autoloading of classes. The value of the "autoload" key is itself a key/value pair. The key psr-4" tells Composer what type of autoloading will be used. The value of this key is, again, itself a key/value pair. What this key/value says: "Start recursively searching for classes identified w/ the namespace ClassSrcFolder_TopLevelNamespace in the directory structure app/ClassSrcFolder/.

The executable code in this project demonstrates this.

  • The three classes are all at different levels of the directory structure.
  • But all are in the same name space!
  • In index.php all that is needed to have them available is to include a use statement constructed of the namespace and the classname.
